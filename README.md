# Student List - Лабораторная работа №10-11

## Описание

Приложение для Android, демонстрирующее возможности Material Design 3.
Оно отображает список студентов с фотографиями, именами, возрастом и описаниями.

## Цель работы:

Научиться применять принципы Material Design в Android-приложениях с использованием Jetpack Compose.
В результате лабораторной работы было создано приложение «Список студентов»,
которое демонстрирует использование цветовых схем, типографики, форм и анимаций Material Design.

## Технологии

- Kotlin
- Jetpack Compose
- Material Design 3

## Функциональность

- Отображение списка из 10 студентов
- Отображение возраста и фотографии студента
- Отображение описания
- Возможность свернуть и развернуть описание
- Поддержка тёмной темы
